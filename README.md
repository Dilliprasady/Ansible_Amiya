# ansible Installation
### system update
```sh
sudo dnf update
```
### Check If pip is installed
```sh
pip --version
pip3 --version
```

### Check If Python is Available, if not install python
```sh
python3 --version
sudo dnf install python3.11
```
### Install pip3
```sh
sudo dnf install python3-pip
pip --version
```

### Install Ansible using pip
```sh
pip install ansible
ansible --version
```
